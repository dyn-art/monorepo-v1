#  Contribution

###  Debug via Jest Test

- [StackOverflow](https://stackoverflow.com/questions/33247602/how-do-you-debug-jest-tests)

1. Start `Javascript Debug` Terminal
2. Set ` Debug` Point
3. Run test via `pnpm run test --watch`, for example:
   ```sh
   pnpm run test -- my-test --watch
   ```
